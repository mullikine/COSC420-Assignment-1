For 4:3:3 network

This is a simple real world data set, called the "iris" set because it is based on iris flowers.
There are 150 I/O pairs, representing 50 examples of each of three species of iris.  
The 4 input values represent measurements of an individual iris, things like leaf width, stem length and so on.
The output (1 0 0, or 0 1 0, or 0 0 1) represents the species of iris.  
You can use this data set to test generalisation: learn many examples of each species, 
then given new inputs see if the network predicts the species correctly.  
Two of the species are fairly similar, a bit difficult to distinguish.
One of the species is more easily distinguished.

Note that this data set is real valued.
